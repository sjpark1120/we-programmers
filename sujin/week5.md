# 4부 미래

## 16장 프로그래밍 언어

- 오늘날 C, C++, 자바, C#, 자바스크립트, 루비, 파이썬, 오브젝트C, 스위프트, 코틀린, 다크, 러스트, 엘름, GO, PHP 등 매우 다양한 언어가 존재
- 시간이 지날수록 새로운 언어에서 근본적인 혁신은 줄어드는 흐름  
  -> 다른 공학 분야처럼 표준 기호와 체계가 정착되는 방향으로 발전 해야 한다
  -> 결국 하나의 언어로 통일될 것 (리스프 계열 언어)

- 타입 시스템
  - 정적 타입 / 동적 타입 각각 장단점 존재
  - 타입 검사를 엄격하게 하되, 그 시점을 런타임으로 미루는 방식에 대한 고민

- 리스프(Lisp)
  - 문법이 극도로 단순하지만 표현력은 매우 풍부
  - 장기적으로 살아남을 가능성이 높다고 평가

- 느낀점
  - 언어는 계속 생기지만 큰 혁신은 점점 줄어드는 느낌
  - 다른 학문처럼 표준화되거나 어느 정도 통일되는 방향도 괜찮아 보임
